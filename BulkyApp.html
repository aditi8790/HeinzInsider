About Heinz-Insider

Heinz-Insider provides the inside scoop into courses at Heinz College in Carnegie Mellon University. Utilizing course data and student generated reviews, we guide you to your career destination as fast as possible. The app is especially useful for new and prospective students. Another great feature is an indicator telling you the time and location of each class to help with schedule planning.

Open Source

Heinz-Insider is now an open source project hosted on Github. This makes it possible for others to make a similar application for other campuses. If you would like to bring the application to your campus, let us know. We may be able to help you collect the data and understand how everything works.
